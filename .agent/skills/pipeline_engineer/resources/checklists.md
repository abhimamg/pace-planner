# Pipeline Engineering Checklists

## Wall Thickness Design Checklist (DNV-ST-F101)
- [ ] Internal Pressure Containment (System Pressure Test) checked? (Sec. 5, D400)
- [ ] Internal Pressure Containment (Operation) checked? (Sec. 5, D400)
- [ ] Local Buckling - External Pressure (Collapse) checked? (Sec. 5, D600)
- [ ] Propagation Buckling checked? (Sec. 5, D700)
- [ ] Material Strength (f_y, f_u) properly derated for temperature?
- [ ] Corrosion Allowance (t_corr) correctly applied?
- [ ] Fabrication Tolerance (t_fab) accounted for?

## On-Bottom Stability Checklist (DNV-RP-F109)
- [ ] Design Metocean data return periods (10-yr, 100-yr) correctly applied?
- [ ] Pipeline OD includes all coatings (anti-corrosion + insulation + weight)?
- [ ] Soil parameters (friction factor, stiffness) based on site-specific data?
- [ ] Static on-bottom stability verified for installation conditions?
- [ ] Dynamic on-bottom stability (Generalized or Absolute) verified for operation?

## Free Span Assessment Checklist (DNV-RP-F105)
- [ ] Seabed profile bathymetry used to determine span geometry?
- [ ] Vortex-Induced Vibration (VIV) - Inline and Cross-flow checked?
- [ ] Fatigue design life (including safety factors) verified?
- [ ] Static stress limits within DNV-ST-F101 criteria?
- [ ] Span correction/mitigation methods (Grout bags, Supports) identified?
